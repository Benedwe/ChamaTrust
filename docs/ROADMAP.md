# Roadmap

## Phase 1: Hackathon Demo

- Mobile dashboard.
- Demo Mobile Money provider adapters.
- Fuji contract deployment.
- Loan voting and execution.
- Advisor insights from deterministic rules.

## Phase 2: Pilot

- Real M-Pesa sandbox integration.
- Webhook reconciliation.
- Admin portal.
- Member KYC.
- Multisig treasury operations.
- SMS and WhatsApp contribution reminders.

## Phase 3: Production

- Regulated stablecoin or tokenized deposit support.
- SACCO compliance reporting.
- Fraud analytics.
- AI credit scoring.
- USSD flows.
- Avalanche Subnet readiness.
